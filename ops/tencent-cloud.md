
# 腾讯云

- 只有上云才能撑住规模化后的发展
    - 初期技术选型上尽可能寻找云支持的
    - 在公司规模小，自建服务基本都做不到 99.999% 高可用
    - 在公司规模发展变迅速时，如果云技术和已有技术契合，迁移成本会低很多很多
